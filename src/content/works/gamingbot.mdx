This project has been through many iterations given its long lifespan, as it once used a Heroku and PostgreSQL database instead of the current JSON one (I am thinking of migrating back to SQL). Because the project started early in my computer science career and lasted this long, many fundamental software engineering practices come from this project, including learning how to update libraries, frameworks, and even the user experience when needed.
<br/>
The bot uses PM2 on the Oracle Cloud instance to run the bot 24/7, and it is currently used in a server with 150 members. Members can interact with the bot via Discord's application command system to utilize a variety of features:
<br/>
- Games (Wordle, Hangman, Slot Machine, Wheel of Fortune)
- Query/Play music in voice chats
- Earn/spend diamonds (the bot's currency)
- Random miscellaneous features:
    - Poll creator
    - Mock messages
    - 8Ball
    - Fetch a user's profile picture
    - Fetch a user's account timestamp
