I have been quite curious of web development recently because it is both programming and art/design: two skillsets that I possessed and was interested in putting together. As my first project in web development, this website gave me the opportunity to learn basic skills in key web development frameworks. The website consists of a combination of HTML, CSS, JavaScript,  TypeScript, Markdown, and MDX due to the involvement of Astro. It is deployed on GitHub Pages using GitHub Actions.
<br/>
My vision for this project from the start was to have a clean, elegant, and minimalistic design built around an interesting, interactive, and dynamic background. In specific, I wanted the background to be a gooey fluid that changes color based on the section the viewport is in. I also wanted some special effects on the mouse, so I settled on a flashlight effect combined with ripples when clicking. The library <a href="https://shaders.com/" target="_blank">Shaders.com</a> is what provides the graphics of the site, which is built upon Three.js. In addition, the library <a href="https://motion.dev/" target="_blank">Motion.dev</a> offers compatibility with Shaders.com and therefore provides some of the keyframing in the background (mainly changing colors and toggling the click effect).
<br/>
Overall I have been having a lot of fun working on this, and I am super proud of the result! Hopefully I will find more web development projects soon.
